# PHP Server for web app under php-fpm

Features ( v.1.2.0)

- http server
- http router
- controller
- actions
- sql repositories  
  repository ( 1 table )  
  composedId ( primary id has more than 1 column )  
  entity ( EAV )  
  Virtual ( 2 and more tables )  
- sql database ( PostgreSql)
- Models and DTOs
- Logger ( with alternative logging )  
  Echo  
  Email  
  File  
- Mailer ( phpmail )  
- http Session
- sitemap
- http view
- multilanguage system
